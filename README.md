# Unity-Source-Tools
Plugin to import resources from the Source engine in Unity3D.

Author(Me): vk.com/lewa_j

VK Public: vk.com/uSrcTools


Suported formats:

bsp - maps

vmt - materials

vtf - textures

mdl - models

vvd - models vertices

vtx - model indices
